# Crescendo_Piano
Team Project for Mobile_Programming(2021)

팀명    : Crescendo

팀장    : 권정우

팀원    : 노성우

주제    : 피아노 연주 애플리케이션


스마트폰의 화면을 직접 터치하여 간단한 피아노, 드럼을 연주하거나 MIDI 컨트롤러에 연결하여 연주할 수 있는 앱을 제작하였습니다. 피아노의 서스테인 페달 기능과 velocity, 옥타브 조절 기능이 포함되어있고, 코드 진행 프리셋 영상을 제작하여 삽입하였습니다.
